# SiriusNLP
Fine tuning ruBERT for Question Answering on SberQuAD

## Docker
Download image:
```
docker image pull yurachinchilla/siriusnlp
```
Run:
```
docker run -d -p 5000:5000 yurachinchilla/siriusnlp
```
Open http://127.0.0.1:5000
